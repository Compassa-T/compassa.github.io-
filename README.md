# Compassa 企業サイト

![Compassa Logo](data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><text y='.9em' font-size='90'>🧭</text></svg>)

**ビジネスを加速させる＋a**

Compassaは、経営者と歯科医院院長のビジネスを支援する企業の公式ウェブサイトです。価格交渉代行、MEO対策、信頼できる業者紹介など、お客様のビジネスを最適化するサービスを提供しています。

## ⚙️ セットアップ方法

1. **プロジェクトのクローン**
```bash
# プロジェクトフォルダに移動
cd compassa-website
```

2. **ブラウザで開く**
```bash
# index.htmlをブラウザで開く
open index.html
# または
start index.html
```

3. **開発サーバーの起動（推奨）**
```bash
# Python 3を使用する場合
python -m http.server 8000

# Node.jsのlive-serverを使用する場合
npx live-server
```

## 🌐 デプロイ方法

### Publishタブからのデプロイ
このプロジェクトをオンラインで公開するには、画面上部の**Publishタブ**をクリックしてください。
ワンクリックでウェブサイトをデプロイし、公開URLを取得できます。

### 手動デプロイ（参考）
以下のプラットフォームでも簡単にデプロイできます：

- **Netlify**: ドラッグ&ドロップで即座にデプロイ
- **Vercel**: GitHubと連携して自動デプロイ
- **GitHub Pages**: 無料でホスティング
